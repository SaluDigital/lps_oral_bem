# OralBem Fortaleza

Landing page estática da OralBem Fortaleza feita com uma stack simples:

- `index.html`
- `styles.css`
- `script.js`

## Como usar

Abra o arquivo `index.html` no navegador ou sirva a pasta com qualquer servidor estático simples.

## Estrutura

- `assets/`: imagens do projeto
- `index.html`: marcação da página
- `styles.css`: estilos globais e responsividade
- `script.js`: menu mobile, barra de progresso e pequenos comportamentos
